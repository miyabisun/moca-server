# キーボードショートカット

管理画面 SPA の「台本」タブでは vim 風のキー操作が使える。入力欄にフォーカス中・
モーダル表示中・IME 変換中・修飾キー併用時は無効。プロジェクト列と台本列の 2 ペインを
`h` / `l` で行き来し、`j` / `k` でカーソルを上下する。
アプリ内でも `?` キー (またはフッターの ? ボタン) で同じ一覧が見られる。

| キー | 動作 |
|---|---|
| `h` / `l` | フォーカス列をプロジェクト / 台本へ移動 (移動先が空なら据え置き) |
| `j` / `k` | カーソルを下 / 上へ (端で停止) |
| `Enter` | プロジェクトを開く / 台本行のアコーディオンを開閉 |
| `Space` | フォーカス中の対象を再生 / 停止 (プロジェクト列＝通し再生、台本列＝単発) |
| `J` / `K` | 台本行を下 / 上へ並び替え |
| `yy` | 台本行をヤンク (コピー) |
| `dd` | 台本行を削除 (ヤンクに退避＝簡易 undo) |
| `p` / `P` | ヤンクした行をカーソルの下 / 上に貼り付け |
| `o` / `O` | 新規行をカーソルの下 / 上に追加 |
| `a` / `i` | タイトル (プロジェクト列) / 台本行のテキスト (台本列) を編集 (キャレット末尾 / 先頭) |
| `m` | 台本行のモードを切替 (アナウンサー ⇄ 演技) |
| `n` | ヘッダーの通知購読メガホンを ON/OFF |
| `?` | ショートカット一覧を表示 (フッターの ? ボタンでも開く) |

`yy` / `dd` は 2 打鍵シーケンス (500ms 以内)。`J`〜`m` は台本列にフォーカスがある時のみ有効。
`Space` はフォーカス列に応じて対象が変わり、`a` / `i` はどちらの列でも編集対象を編集する。
`n` / `?` はどちらの列でも効く。
